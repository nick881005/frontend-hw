# frontend-hw

前端angular框架

先啟動intellij , 打開vs code在終端機輸入 ng serve -- open以進行聯通

點擊get user可以得到所有user資料

點擊create user可以新增使用者

只是刪改按件功能做不出來只能顯示圖示
